# `012` tens digits

## 📝 Instructions:

1. Complete the `tens_digit()` function. Given an integer, `tens_digit()` returns its tens digit.

## 📎 Example 1:

```py
output = tens_digit(1234)
print(output)  # 3
```

## 📎 Example 2:

```py
output = tens_digit(179)
print(output)  # 7
```

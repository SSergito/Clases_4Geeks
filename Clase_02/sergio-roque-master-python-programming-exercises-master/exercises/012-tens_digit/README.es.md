# `012` tens digits

## 📝 Instrucciones:

1. Completa la función `tens_digit()` para que dado un número entero, retorne sus decenas.

## 📎 Ejemplo 1:

```py
output = tens_digit(1234)
print(output)  # 3
```

## 📎 Ejemplo 2:

```py
output = tens_digit(179)
print(output)  # 7
```
